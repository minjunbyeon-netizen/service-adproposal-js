## 복원·확장 방법

1. **사이드패널 선택**: localhost:8881 → 해당 V## PT 클릭
2. **여러 버전 결합**: 특정 버전을 base로 삼아 새 clone → 다른 버전의 페이지 import
3. **GitHub Pages 교체**: `scripts/regen_pt.py`에서 PID 변경 → regen → push

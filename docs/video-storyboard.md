# 영산대학교 와이즈유 - 영상 스토리보드

## 캠페인 개요

**클라이언트:** 영산대학교 (브랜드명: 와이즈유 / WISE University)
**메인 콘셉트:** "2지망에서 1지망으로 나갔다"
**핵심 강점:** 
- 항공서비스학과 취업률 96.4%
- 뷰티디자인학과 94.1%
- 경찰행정학과 93.7%

**제작 원칙:**
1. 배우 기용 금지 - 실제 졸업생/재학생만 출연
2. 드론샷, BGM, 내레이션 공식 금지 - 인터뷰 다큐 형식
3. "고백의 구조" - 불편한 진실 인정 후 반전
4. 팩트만 전달 - 감동 강요 금지, 눈물·웃음 연출 금지

**예산:** 25,000,000원/년

### 음악/사운드 방향

**0-5초:** 차분한 피아노 (슬픔 아님, 신중함)
**5-54초:** 점진적으로 현악기 추가 (바이올린, 첼로), 템포 유지
**54-60초:** 피아노+현악 풀 사운드, 점진적 페이드아웃

**사운드 이펙트:**
- 자막 등장 시 아주 미세한 타이핑 음 또는 무음 (연출 금지)
- 화면 전환 시 부드러운 트랜지션 (효과음 사용 금지)

**배경 소음:**
- 인터뷰 영상 안 환경음은 살려둠 (사무실 미세한 움직임, 숨소리)
- 과도한 청소 또는 제거 금지

**대사:**
"처음엔 솔직히 '영산대?'라는 생각이 들었어요. 아이가 2지망이라고 했을 때..."

# 와이즈유(영산대학교) 광고 시안 4종

> 지면 광고 / 인쇄 광고 콘셉트
> 
> 클라이언트: 영산대학교 (브랜드명: 와이즈유 WISE University)
> 위치: 부산 해운대
> 핵심 강점: 항공서비스학과 96.4% 취업률, 뷰티디자인학과 94.1%, 경찰행정학과 93.7%
> 메인 콘셉트: "2지망에서 1지망으로 나갔다"

---

## 개발 원칙

1. **기존 문법 탈피** -- 대학 광고의 관례(드론 샷 + 웃는 학생 + 추상 슬로건)를 완전히 버림
2. **텍스트 중심** -- 레이아웃과 장식 없음. 말 자체가 디자인
3. **질문형 카피** -- 수험생이 스스로 답하게 만드는 구조
4. **숫자는 증거** -- 주장이 아니라 검증된 데이터

## 시안 1: 질문형 (불편한 질문으로 멈추게 하는)

### 타깃
고등학교 3학년 수험생 / 재수생

### 판형
신문 전체 페이지 (신문 전15단 기준)

### 디자인 레이아웃 (글 설명)

**배경**: 흰색 (텍스트 가독성 최우선)

**메인 영역**:
- 페이지 상단부터 시작
- 메인 질문 카피가 가운데 세로로 배치됨
- 폰트: Roboto + Noto Sans KR, 700 굵기, 48pt 이상
- 색상: 검정색 (#000000)
- 줄바꿈 최소화, 한 줄에 3-5 글자씩 끊어서 시선 흐름 만들기

**서브 영역**:
- 메인 카피 하단 여백 40px
- 서브 설명 카피 (14pt, 400 굵기)
- 검정색 텍스트

**로고 영역**:
- 페이지 우측 하단 또는 좌측 하단
- 와이즈유 로고 + "영산대학교" 텍스트 병행
- 크기: 로고만 100x100px 내외

**액센트**:
- 메인 카피에 정확히 1개 단어/구절을 #0071E3 파란색으로 강조
- 예: "미래" 단어만 파란색 처리

### 메인 카피 (원문)

```
그 대학 떨어졌어도
괜찮은가.

더 나은 선택지가 있다는 걸
모르니까.
```

### 서브 카피 (원문)

```
2지망에서 1지망으로 나갔다.

항공서비스학과 96.4% 취업률
뷰티디자인학과 94.1% 취업률
경찰행정학과 93.7% 취업률

와이즈유가 정답입니다.

www.wiseu.ac.kr
```

### 작동 원리

질문의 형태로 독자의 마음 속 불안감을 직접 건드림. 이미 합격장을 품고 있는 수험생도 "정말 그 대학이 최선인가"라는 의심의 씨앗을 심어줌. 부정적 질문이 아니라 희망의 재검토로 기능함.

## 시안 2: 숫자형 (압도적 수치 1개로 시선 잡는)

### 타깃
학부모 / 입시 관계자 / 교사

### 판형
잡지 전면 광고 (가로 210mm x 세로 297mm, A4 기준)

### 디자인 레이아웃 (글 설명)

**배경**: 흰색

**메인 영역**:
- 페이지 중상단 (상단부터 약 40% 지점)
- 압도적 숫자 "96.4%" 중앙 배치
- 폰트: Roboto 700 굵기, 120pt 이상 (매우 큼)
- 색상: #0071E3 파란색 (브랜드 액센트)
- 단순함 강조

**설명 텍스트 영역**:
- 숫자 하단 20px 여백
- "항공서비스학과 취업률" (18pt, 검정색)
- 그 아래 여백 30px

**신뢰도 영역**:
- 페이지 하단 1/4 영역
- 세 개의 숫자를 가로로 나열 (ROW 형태)
- "96.4% | 94.1% | 93.7%"
- 각 숫자 아래 학과명 (한 줄)
- 폰트: 14pt, 400 굵기

**로고 및 URL**:
- 우측 하단
- 로고 + "와이즈유 WISE University"
- "www.wiseu.ac.kr | 대학 문의: 051-****-****"

### 메인 카피 (원문)

```
96.4%
```

(그것이 전부. 숫자만 압도한다)

### 서브 카피 (원문)

```
항공서비스학과 취업률

가장 높은 취업률을 자랑하는 와이즈유.

최고의 교육.
최고의 결과.

와이즈유 WISE University
```

### 작동 원리

96.4% 한 개 숫자가 페이지를 지배함으로써 어떤 설명보다 강력한 메시지를 전달. 이 숫자 자체가 "와이즈유는 신뢰할 수 있다"는 증거로 작동하며, 학부모의 "우리 자식 취업이 보장되나"라는 근본적 불안을 해소.

## 시안 3: 대비형 (입학 전 vs 졸업 후 반전)

### 타깃
전체 (수험생 / 학부모 / 교사)

### 판형
버스 쉘터 광고 (가로 1200mm x 세로 1000mm 또는 표준 120x100cm)

### 디자인 레이아웃 (글 설명)

**배경**: 흰색

**레이아웃 분할**:
- 좌측 절반 (50%): 검정색 배경
- 우측 절반 (50%): 흰색 배경
- 명확한 경계선 없음 (색상 대비만으로 구분)

**좌측 영역 (검정 배경)**:
- 텍스트 색상: 흰색
- 메인 카피 "입학 전" 배치
- 폰트: Roboto 700, 60pt 이상
- 가운데 정렬, 세로 중앙 배치
- 부가 설명: 작은 글씨 (18pt), 흰색

**우측 영역 (흰색 배경)**:
- 텍스트 색상: 검정색
- 메인 카피 "졸업 후" 배치
- 폰트: Roboto 700, 60pt 이상
- 가운데 정렬, 세로 중앙 배치
- 부가 설명: 작은 글씨 (18pt), 검정색
- 1개 문구는 #0071E3 파란색 강조

**하단 영역**:
- 양쪽 배경을 가로질러 배치
- 중앙 정렬, 로고 + URL
- 폰트: 14pt
- "와이즈유 WISE University | www.wiseu.ac.kr"

### 메인 카피 (원문)

**좌측 (검정 배경, 흰 텍스트)**:
```
고민만 많고
실력은 없었다
```

**우측 (흰 배경, 검정 텍스트)**:
```
96%의 동기들이
취업했다
```

### 서브 카피 (원문)

**좌측**:
```
그저 대학생인 줄만 알았는데
```

**우측**:
```
와이즈유는 달랐다
```

**하단 (공통)**:
```
2지망에서 1지망으로 나갔다.
와이즈유 WISE University
```

### 작동 원리

좌우 색상과 메시지의 대비가 심리적 반전 효과를 만듦. 처음엔 부정적 자화상(고민만 많고)으로 시작했다가 우측에서 긍정적 결과(96% 취업)로 반전되며, 버스를 기다리는 사람들의 시선을 좌에서 우로 자연스럽게 유도. 신분증처럼 명확한 흑백 구분으로 기억에 오래 남음.

## 시안 4: 도발형 (기존 대학 광고 정면 비판)

### 타깃
고등학교 3학년 수험생 / 재수생 (진취적 성향)

### 판형
신문 전체 페이지 (신문 전15단 기준)

### 디자인 레이아웃 (글 설명)

**배경**: 흰색

**상단 영역 (메인 도발)**:
- 페이지 상단 20% 차지
- 진한 검정색 (거의 검정): #1a1a1a
- 메인 카피 배치 (좌측 정렬, 들여쓰기 없음)
- 폰트: Roboto 700, 40pt
- 줄바꿈으로 문장의 리듬감 극대화

**중단 영역**:
- 상단부터 약 40-80% 구간
- 흰색 배경, 검정 텍스트
- 세로줄 3개 ROW로 대학 선택 항목 나열
- 각 항목: 14pt, 400 굵기
- 좌측 정렬, 충분한 가로 여백
- 특정 항목 하나(와이즈유)는 #0071E3 파란색 강조

**하단 영역**:
- 페이지 하단부터 20%
- "2지망에서 1지망으로 나갔다" 문구
- 취업률 3개 숫자 (ROW)
- 로고 + URL

**액센트 라인**:
- 상단과 중단 사이 가로줄 1px (검정색) 구분선

### 메인 카피 (원문)

```
드론 샷으로 찍은 교정.
웃는 학생들.
추상적인 슬로건.

이제 이런 광고는 끝이다.
```

### 서브 카피 (원문)

```
진짜 질문: 졸업 후 뭐 할 거냐.

1. 스펙 좋은 대학 나왔는데
   일자리는 찾지 못했다

2. 그럭저럭 평범한 곳 나왔는데
   96.4%가 1년 안에 취업했다

당신의 선택은.

와이즈유를 선택했다.
항공서비스학과 96.4% 취업률
뷰티디자인학과 94.1% 취업률
경찰행정학과 93.7% 취업률

www.wiseu.ac.kr
```

### 작동 원리

기존 대학 광고 관례를 명시적으로 비판함으로써 "이 학교는 다르다"는 신뢰감을 형성. 거짓 선택지(스펙 좋은데 일자리 없음 vs 평범한데 취업률 높음)를 제시해 독자가 논리적으로 스스로 와이즈유를 선택하게 유도. 도발적 톤이 "이 대학은 학생을 우롱하지 않는다"는 메시지 전달.

## 기대 효과

1. **기억 효과** -- 기존 광고와 완전히 다른 톤으로 경쟁력 있는 학교로 인식
2. **신뢰도 증가** -- 추상적 슬로건 대신 구체적 취업률 수치 제시로 신뢰성 확보
3. **타겟층 분산** -- 4개 시안으로 수험생/학부모/교사 모두 커버
4. **미디어 추천** -- 톤과 형식이 다르면 언론사의 피처 대상이 될 가능성 증가

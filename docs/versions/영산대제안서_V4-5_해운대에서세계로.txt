# 컨셉: 해운대에서 세계로
----------------------------------------
# 해운대에서 세계로

서울이 아니라서 약한 게 아니다. 해운대라서 강하다.
QS 55위 + 해운대 = 세계가 인정한 해변 도시의 대학.
부산 로컬 프라이드를 최강 자산으로 전환.

# 슬로건 & 캐치
----------------------------------------
## 슬로건: 해운대발

## 캐치프레이즈: 세계가 인정한 해변의 대학

출발지가 해운대라는 것이 약점이 아니라 도착지가 세계라는 증거.

# 캠페인: FROM HAEUNDAE
----------------------------------------
# FROM HAEUNDAE / 해운대발

## Phase 1: 출발 (6~8월)
"출발지: 해운대." 공항 출발 게이트 비주얼. 어디로 가는지 안 알려줌.

## Phase 2: 도착 (9~11월)
"도착지: 세계 55위." 졸업생이 일하는 세계 각지 장면.

## Phase 3: 탑승 (12~2월)
"탑승권 발급 중." 원서 접수 = 탑승권. 행동 유도.

# 지면 시안
----------------------------------------

## 시안 A: 탑승권
----------------------------------------
실제 항공 탑승권 디자인.
FROM: HAEUNDAE / TO: WORLD TOP 55
출발: 2026 / 도착: 4년 후 취업률 96.4%

탑승권이라는 익숙한 포맷이 여행/항공 이미지와 연결.

## 시안 B: 지도
----------------------------------------
부산 해운대에서 세계로 뻗는 항공 노선도.
각 도착점에 졸업생 취업처 표기.
"서울이 아니라 해운대. 그래서 세계."

# 영상
----------------------------------------

## 60초: 해운대발 세계행
----------------------------------------
해운대 해변 새벽 -> 공항 -> 기내 -> 세계 각지 졸업생 직장.
내레이션: "출발지는 해운대였습니다."
엔딩: "도착지는 세계. 와이즈유."

## 15초: 좌표
----------------------------------------
GPS 좌표 숫자가 움직인다. 해운대(35.1N) -> 도쿄 -> 두바이 -> 파리.
"출발지가 어디든, 도착은 같다. QS 55위."

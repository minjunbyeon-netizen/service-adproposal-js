# 틀을 깨는 대학

3개 관점 통합:
- 머스크: 서열은 입력값, 결과는 출력값. 출력값으로 재정의.
- 잡스: 현명함이란 남들이 보지 않는 곳에서 답을 찾는 감각.
- 오길비: 팩트가 편견보다 강하다. 읽으면 안다.

대학 서열이라는 틀 자체를 부수는 접근.
"2지망"이라는 프레임을 거부하고, WISE = 기존 상식에 휘둘리지 않는 판단력.

# 지면 시안 (관점별 4종 x 3)
----------------------------------------

## 머스크 시안: 검색창 / 성적표 vs 성과표 / 세계지도 / 지원서
----------------------------------------
## 머스크 시안 4종

**검색창**: 검색어란에 "영산대 취업률". 결과: 96.4%. 수험생 실제 행동을 시안에 재현.

**성적표 vs 성과표**: 좌측 수능 성적표(흐릿) / 우측 졸업생 성과표(선명). 입구 vs 출구.

**세계지도**: 부산에서 세계로 뻗는 항공 노선도. 각 도착점에 졸업생 취업처.

**지원서**: 실제 원서 양식 위에 와이즈유 로고 + 핵심 수치 3개. 행동 유도.

## 잡스 시안: 빈 액자 / 거꾸로 순위표 / 두 개의 문 / 졸업장
----------------------------------------
## 잡스 시안 4종

**빈 액자**: 미술관 흰 벽, 빈 액자 하나. "남들이 안 보는 곳에 답이 있다." 여백의 힘.

**거꾸로 순위표**: 대학 순위표가 뒤집혀 있다. 맨 아래(실제 맨 위)에 와이즈유 성과. 물리적 전복.

**두 개의 문**: 간판의 문 vs 결과의 문. "어느 문을 여시겠습니까." 학부모 합리적 판단 자극.

**졸업장 클로즈업**: "4년 후, 이 이름이 말해준다." 최종 산출물에 집중.

## 오길비 시안: 사전 정의 / 긴 카피 / 오답 교정 / 비교표
----------------------------------------
## 오길비 시안 4종

**사전 정의 수정**: 국어사전 "지방대" 항목에 빨간 삭제선. 새 정의: 세계가 인증한 대학. 권위감+편견 삭제.

**긴 카피**: "이 광고를 끝까지 읽을 사람만 지원하라." 800자 팩트 나열. 읽는 행위가 필터링이자 설득.

**오답 교정**: 시험지 스타일. 오답: "대학은 순위대로." 정답: "96.4%가 취업하는 대학이 있다."

**비교표**: 좌측 "지방 사립대 평균 65%" vs 우측 "와이즈유 96.4%". 팩트 병렬. 독자가 스스로 결론.

# 영상 스토리보드 (관점별)
----------------------------------------

## 머스크: 공장 리셋 / 역추적 / 오답노트
----------------------------------------
## 머스크 영상 3종

**공장 리셋 (15초)**: 서열 랭킹 화면이 글리치되며 깨진다. 리셋. "QS 55위, 취업률 96.4%"만 남는다.

**역추적 (30초)**: 호텔 총지배인 로비 -> 시간 역추적 -> 면접-졸업-캠퍼스-원서. "시작이 달랐다."

**오답노트 (60초)**: 오답 목록: "대학은 서열순" "지방대는 한계" "취업은 간판". 하나씩 X표. 마지막: 와이즈유 팩트 3개.

## 잡스: 질문 / 발견 / 4년 후
----------------------------------------
## 잡스 영상 3종

**질문 (6초)**: 검은 화면. "왜 그 순서대로?" 3초 정적. 로고만. 무음.

**발견 (30초)**: 서점에서 대학 가이드북 넘기다 바닥에 떨어진 페이지 "QS 55위". 눈빛 변화. "남들이 안 보는 곳에 답이 있다."

**4년 후 (60초)**: 4분할. 4명 졸업생 각자 현장. 내레이션 없이 현장음만. 마지막에 4명이 카메라를 본다. "현명한 선택은 4년 뒤에 증명된다."

## 오길비: 5가지 사실 / 편견 인터뷰 / 읽는 시간
----------------------------------------
## 오길비 영상 3종

**5가지 사실 (15초)**: 검은 화면. 5개 팩트 타이핑. "어느 대학인지 아직 안 말했다." 와이즈유.

**편견 인터뷰 (30초)**: 거리 인터뷰 "영산대 아세요?" "아니요." 팩트 제시. 표정 변화. "편견에는 유통기한이 있다."

**읽는 시간 (60초)**: 수험생이 와이즈유 페이지에서 멈춘다. 1분간 읽는 모습. 하단에 팩트 자막. "읽으면 안다."

# 컨셉: 지혜가 실력이다
----------------------------------------
# 지혜가 실력이다

WISE = 지혜. 기존 "지혜로운 가치를 배우는 대학"이 추상적이었다면,
V4-10은 지혜 = 실전적 판단력 = 취업으로 재정의.
부엉이(지혜의 상징) 마스코트와 연결.
건학이념의 깊이 + 현대적 결과의 결합.

# 슬로건 & 캐치
----------------------------------------
## 슬로건: 지혜가 실력이다

## 캐치프레이즈: 아는 것과 되는 것은 다르다

"아는 것"(지식)은 어디서든 배운다. "되는 것"(실력)은 와이즈유에서 만들어진다.

# 캠페인: WISE WORKS
----------------------------------------
# WISE WORKS / 지혜가 일한다

## Phase 1: 정의 (6~8월)
"지혜란 무엇인가." 추상적 질문을 던진다. 부엉이 마스코트 활용.

## Phase 2: 증명 (9~11월)
"지혜가 만든 숫자." 96.4%, 55위, 25명. 지혜의 결과물.

## Phase 3: 선언 (12~2월)
"지혜가 실력이다." 와이즈유 선택 = 지혜로운 선택.

# 지면 시안
----------------------------------------

## 시안 A: 재정의
----------------------------------------
흰 배경. 중앙에 단 한 글자:

> 지혜.

120pt. 그 아래 18pt:
"96.4%를 만든 힘. 와이즈유."

추상을 구체로 재정의하는 시안.

## 시안 B: 지식 vs 지혜
----------------------------------------
분할 구성:
좌: "지식: 배운 것" (회색, 흐릿)
우: "지혜: 해낸 것" (검정, 선명)

하단: "항공서비스 96.4%. 호텔 총지배인 25명.
지혜가 실력이다. 와이즈유."

# 영상
----------------------------------------

## 30초: 부엉이의 눈
----------------------------------------
부엉이 마스코트 눈 클로즈업에서 시작.
부엉이 눈에 비친 장면: 캠퍼스 -> 실습실 -> 졸업식 -> 직장.
"지혜의 눈으로 본 4년. 와이즈유."

## 60초: 원융무애
----------------------------------------
건학이념 브랜드 필름. 나와 세계의 경계를 넘는 졸업생들.
해운대 -> 세계 호텔 -> 하늘 위 -> 경찰서.
"경계 없는 지혜. WISE University."

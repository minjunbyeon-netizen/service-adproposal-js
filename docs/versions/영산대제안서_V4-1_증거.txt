# 컨셉: 증거의 대학
----------------------------------------
# 증거의 대학

와이즈유는 약속을 파는 대학이 아니라 증거를 파는 대학이다.

기존: 감성 -> 약속 -> 신뢰? (검증 불가)
와이즈유: 숫자 -> 증거 -> 신뢰. (검증 완료)

**주장하지 마라, 증명하라.**

# 캠페인: PROVE IT.
----------------------------------------
# PROVE IT.

대학이 증명하는 게 아니다. 졸업생이 증명한다.

## Phase 1: Tease (6~7월)
숫자만 던진다. 설명 없음. "96.4%" / "세계 8위" / "25명"

## Phase 2: Launch (9~10월)
숫자에 얼굴을 붙인다. 졸업생 실명+직장+인용.

## Phase 3: Sustain (11~1월)
"PROVE IT. 증명은 우리가 아니라 당신이 한다."

# 영상 스토리보드
----------------------------------------

## 메인 60초
----------------------------------------
## 메인: "2지망에서 1지망으로 나갔다"

씬1: 검은화면 "솔직히 말하겠습니다" (무음)
씬2: 승무원 졸업생 "1지망은 아니었습니다"
씬3: 뷰티 졸업생 "나만 부산이었어요" (94.1%)
씬4: 경찰 졸업생 "40명 중 38명 경찰" (93.7%)
씬5: 해운대 캠퍼스 드론 (QS 55위/세계 8위)
씬6: 호텔 총지배인 "25명입니다"
씬7: 3인 교차 "여기 오길 잘했다"
씬8: "2지망에서 1지망으로 나갔다" + 로고

## 숏폼 15초 x 3 + 학부모 90초
----------------------------------------
## 숏폼 3종

항공: "동기 96%가 하늘에 있더라고요"
뷰티: "면접 때 '영산대요?' 하던 사람들이 이제 저한테 메이크업 맡깁니다"
경찰: "40명 중 38명이 경찰. 나머지 2명은 소방관"

## 학부모 90초

"그때는 걱정했는데, 지금은 자랑합니다"
어머니 인터뷰: 걱정 -> 지켜봄 -> "동창회에서 먼저 얘기해요"
